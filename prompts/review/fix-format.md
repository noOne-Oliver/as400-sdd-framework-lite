# Prompt: Fix Format

> 用于自动修复代码格式问题，不改变任何逻辑。

## 任务
对以下 RPGLE 代码进行格式修复。

## 原始代码
```rpgle
[粘贴需要修复的代码]
```

## 修复规则

### 1. 缩进规范
- 统一使用 2 空格缩进
- 不要使用 Tab

### 2. 关键字大写
```rpgle
// ✅ 正确
IF %FOUND;
  DOW NOT %EOF;
  ENDDO;
ENDIF;

// ❌ 错误
if %found;
  dow not %eof;
  enddo;
endif;
```

### 3. 字符串引号
- 统一使用单引号 `'text'`
- 不要使用双引号 `"text"`

### 4. 结构完整性
- 确保所有 IF/DO/FOR 有对应的 END
- 确保 DCL-PROC / DCL-PI 配对

### 5. 空格规范
- 操作符前后各一个空格：`a = b` 而非 `a=b`
- 逗号后一个空格：`CALL 'PGM' PARM(a: b: c)`
- 移除行尾多余空格
- 连续空行不超过 1 行

### 6. 注释规范
- 单行注释使用 `// 注释`
- 多行注释使用 `/* 注释 */`
- 注释放在代码上方或行尾

## 输出格式
```rpgle
[修复后的代码]
```

## 注意事项
- ❌ 不要改变任何代码逻辑
- ❌ 不要重新排列代码顺序
- ❌ 不要修改变量名或字段名
- ❌ 只做格式修复，不做功能修改
