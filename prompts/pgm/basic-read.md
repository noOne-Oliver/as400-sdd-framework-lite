# Prompt: 读取文件程序

## 任务
生成一个 RPGLE 程序，从 [文件名] 读取数据。

## 输入参数
| 参数 | 类型 | 说明 |
|------|------|------|
| 参数1 | CHAR(10) | 说明 |
| 参数2 | PACKED(15:2) | 说明 |

## 查询条件
- 字段A = [条件值]
- 字段B > [条件值]

## 输出
- 成功时返回记录列表（字段1, 字段2, 字段3）
- 失败时返回错误码和消息

## 上下文
- 文件定义：见 /inc/[文件名]_D.rpgle
- 代码风格参考：/src/PGM[参考程序].rpgle

## 要求
1. 使用 FREE 格式 RPGLE
2. 使用 EXTNAME 引用文件
3. 包含 INFSPC 错误处理
4. 所有字段引用必须与文件定义一致
5. 使用 DCL-PRC 定义过程接口

## Copilot 注意事项 ⚠️
- [ ] 使用 "字段名" 双引号引用
- [ ] 字符字段处理 %TRIM
- [ ] CHAIN 前先判断文件状态
- [ ] 错误记录到 ERRLOG 或 job log

## 输出格式
直接输出完整代码。
